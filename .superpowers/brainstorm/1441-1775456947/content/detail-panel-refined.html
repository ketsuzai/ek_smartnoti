적용된 디자인시스템 수정 내역

  • 폰트 8~10px → 최소 11px(태그), 12px(보조), 13px(본문), 14px(헤더)
  • 버튼 패딩 2~3px → 5~7px 10~14px으로 확대
  • 버튼 border-radius → 6px (radius-sm 기준)
  • 토글 라벨 오른쪽 정렬 — 식사구분 라벨과 분리
  • 토글 ON 시 해당 식사구분 섹션 상단 2px 액센트 보더 + 배경색 강조
  • + 등록 버튼 → #0EA5E9 solid (운영관리 액센트 적용)
  • 카드 radius → 12px (radius-lg), 반 서브카드 → 8px
