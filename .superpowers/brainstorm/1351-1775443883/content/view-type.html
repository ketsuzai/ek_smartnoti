A. 월간 캘린더 뷰

장점

  • 한눈에 월간 현황 파악
  • 학부모 앱 뷰와 동일감

단점

  • 셀이 작아 메뉴 내용 축약
  • 알레르기 정보 공간 부족

B. 주간 테이블 뷰

1주 단위 행·열 테이블. 행=식사구분(점심/간식), 열=요일. 주 단위 네비게이션.

장점

  • 메뉴·알레르기 정보 충분히 표시
  • 입력·수정이 직관적

단점

  • 월간 전체 흐름 파악 어려움
  • 주 이동이 잦아질 수 있음

C. 하이브리드 (미니 캘린더 + 상세 패널)

상단에 미니 월 달력(날짜 선택용), 하단에 선택된 날짜의 상세 식단 패널. 두 정보를 한 화면에.

장점

  • 월 전체 조망 + 일별 상세 동시에
  • 메뉴 텍스트 넉넉히 표시 가능

단점

  • 레이아웃 복잡도 높음
  • 한 번에 하루만 볼 수 있음
