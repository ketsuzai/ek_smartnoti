# 17-MEMBER-APPROVAL.md — 멤버/승인

> 상태: ✅ 완료 (2026-04-13)
> 기존 원아관리(operation-child.html)를 대체·확장하는 통합 멤버 관리 화면
> 파일: `src/pages/oper/operation-member.html`

---

## 개요

기관에 속한 **교사 멤버**와 **원아 멤버**를 통합 관리하는 화면.
- 교사 멤버 수정은 기관관리자만 가능
- 교사 본인은 자신의 정보 조회만 가능

---

## 구현 완료 스펙

### 1차 탭
| 탭 | 설명 | 권한 |
|----|------|------|
| 가입승인 | pending 신청 카드 리스트 + 승인/거부 플로우 | org_admin/super만 |
| 멤버관리 | 교직원/원아 2차 서브탭 | 전체 |

### 2차 서브탭 (멤버관리 내)
| 서브탭 | 컬럼 | 드로어 탭 |
|--------|------|-----------|
| 교직원 | 이름·역할배지·담당반·상태·가입일·최근접속·관리 | 기본정보/담당반 |
| 원아 | 상태·아동명·성별·만나이·반·주보호자·보호자수·앱연결·입원일·관리 | 기본정보/보호자/건강특이사항/활동이력 |

### KPI 카드 (4개)
전체멤버(sky) | 가입대기(amber, 클릭→가입승인탭) | 교직원(navy) | 원아(green)

### 가입승인 카드 UI
- `.approval-card` 컴포넌트: 아바타 + 이름·역할배지·연락처·연결원아 + 날짜 + [거부][승인] 인라인 버튼
- 필터칩: 전체 / 교직원 / 원아 보호자
- 승인 → confirm 모달 → 카드 사라짐 + KPI 업데이트
- 거부 → 사유 입력 모달 (비워두면 toast 에러)

### 역할별 권한
| 기능 | org_admin/super | teacher |
|------|:---:|:---:|
| 가입승인 탭 | ✅ | ❌ (숨김) |
| 교직원 수정 | ✅ | ❌ |
| 원아 보호자 연결 | ✅ | ❌ |
| 원아 삭제 | ✅ | ❌ |
| 원아 상태변경 | ✅ | ✅ |

### 역할배지 스타일
- `.r-org_admin` (퍼플) / `.r-teacher` (블루)
- `.s-active` (그린) / `.s-inactive` (그레이)

## 탭 구조

| 탭 | 대상 | 기관관리자 | 교사관리자 |
|----|------|-----------|-----------|
| 원아 멤버 | 등록된 원아 목록 | 조회·수정·삭제 | 담당 반 조회만 |
| 교사 멤버 | 등록된 교사 목록 | 조회·수정·삭제 | 조회만 |

---

## 원아 멤버 기능 (기존 operation-child.html 이전)

- 반별 탭 필터 + 상태/검색 필터바
- 원아 목록 테이블 → 우측 드로어 (상세·보호자·수정)
- 원아 등록·상태변경 모달 / 보호자 연결 관리

---

## 교사 멤버 기능 (신규)

| 기능 | 설명 |
|------|------|
| 교사 목록 | 이름·역할·담당반·상태 표시 |
| 교사 상세 드로어 | 연락처, 담당 반, 계정 상태 조회 |
| 교사 수정 (기관관리자만) | 담당 반 변경, 계정 활성/비활성 |
| 교사 초대 연동 | 초대장 관리 페이지로 이동 버튼 |
