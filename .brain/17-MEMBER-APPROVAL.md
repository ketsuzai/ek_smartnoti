# 17-MEMBER-APPROVAL.md — 멤버/승인

> 상태: 🔲 미시작 (준비중)
> 기존 원아관리(operation-child.html)를 대체·확장하는 통합 멤버 관리 화면

---

## 개요

기관에 속한 **교사 멤버**와 **원아 멤버**를 통합 관리하는 화면.
- 교사 멤버 수정은 기관관리자만 가능
- 교사 본인은 자신의 정보 조회만 가능

---

## 파일

- `src/pages/oper/operation-member.html`

---

## 탭 구조

| 탭 | 대상 | 기관관리자 | 교사관리자 |
|----|------|-----------|-----------|
| 원아 멤버 | 등록된 원아 목록 | 조회·수정·삭제 | 담당 반 조회만 |
| 교사 멤버 | 등록된 교사 목록 | 조회·수정·삭제 | 조회만 |

---

## 원아 멤버 기능 (기존 operation-child.html 이전)

- 반별 탭 필터 + 상태/검색 필터바
- 원아 목록 테이블 → 우측 드로어 (상세·보호자·수정)
- 원아 등록·상태변경 모달 / 보호자 연결 관리

---

## 교사 멤버 기능 (신규)

| 기능 | 설명 |
|------|------|
| 교사 목록 | 이름·역할·담당반·상태 표시 |
| 교사 상세 드로어 | 연락처, 담당 반, 계정 상태 조회 |
| 교사 수정 (기관관리자만) | 담당 반 변경, 계정 활성/비활성 |
| 교사 초대 연동 | 초대장 관리 페이지로 이동 버튼 |
