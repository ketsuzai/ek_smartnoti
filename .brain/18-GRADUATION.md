# 18-GRADUATION.md — 진급/졸업

> 상태: 🔲 미시작 (준비중)

---

## 개요

원아의 연말 진급(반 이동) 및 졸업 처리를 일괄로 수행하는 화면.
연 1회 또는 필요 시 수동으로 실행하는 배치성 작업.

---

## 파일

- `src/pages/oper/operation-graduation.html`

---

## 핵심 기능 (초안)

| 기능 | 설명 |
|------|------|
| 진급 시뮬레이션 | 현재 반 구성 기준으로 다음 연도 반 배정 미리보기 |
| 진급 처리 | 원아별 또는 반 단위로 다음 반으로 이동 |
| 졸업 처리 | 최고 연령반 원아를 졸업 상태로 일괄 전환 |
| 졸업생 보관 | 졸업 원아 데이터를 보관 상태로 유지 (삭제 X) |
| 반 초기화 | 새 학년도 시작 시 반 구성 리셋 |
| 이력 조회 | 진급/졸업 처리 이력 타임라인 |

---

## 권한

| 역할 | 접근 | 실행 |
|------|------|------|
| 기관관리자 | O | O |
| 교사관리자 | X | X |

---

## 주의사항

- 진급/졸업 실행은 되돌리기 어려운 작업 → 확인 모달 필수
- 졸업 원아 데이터는 소프트 삭제 (삭제 플래그만 세팅)
