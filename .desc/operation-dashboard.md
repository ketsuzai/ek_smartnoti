## 화면 1: 대시보드 — 기관관리자 뷰

> 기관관리자 또는 통합관리자가 기관관리자 컨텍스트로 접속한 경우.
> 반 필터 없음. 전체 9개 반 기준으로 기관 현황 표시.

---

**① 페이지 헤더**

- 좌측: 기관명 (예: "해맑은 어린이집"), 부제 "오늘 기관 현황을 한눈에 확인하세요."
- 우측: 오늘 날짜 칩 (예: "2026년 3월 31일 화")
- 반 선택 필터바: **기관관리자 뷰에서는 표시하지 않음** (항상 전체 반 기준)

---

**② KPI 카드 (4개)**

| 순서 | 항목 | 수치 예시 | 보조 텍스트 |
|------|------|-----------|------------|
| 1 | 전체 원아 수 | 140명 | 총 9개반 |
| 2 | 교사 수 | 12명 | 담임 8 · 보조 4 |
| 3 | 오늘 알림장 | 56건 | 전체 84건 중 |
| 4 | 학부모 접속률 | 71% | ▲ 3% 전일 대비 |

- 4개 카드가 동일 너비 1행으로 나열
- 각 카드: 좌측 아이콘 박스 + 우측 (항목명 / 수치 / 보조텍스트)
- 전체 원아 수는 전체 반 CHILDREN_DATA 합산으로 동적 계산

---

**③ 반별 현황 카드 (메인 카드, 2fr 너비)**

- 카드 제목: "반별 현황" (우측 액션 버튼 없음)
- 내부: 2열 그리드로 반 카드 나열, 총 9개 반
- **반 카드 7개 이상 시**: 카드 본문 영역에 `max-height: 440px` + 세로 스크롤 적용

각 반 카드 구성:

| 영역 | 내용 |
|------|------|
| 상단 | 반 이름 (굵게) + 연령 (작은 텍스트, 예: 만5세) |
| 중단 | 담임 이름 · 보조 이름 (예: "담임 이하늘 보조 김보라") |
| 하단 | 원아 / 출석 / 투약 3개 수치 박스 |

수치 박스 색상 규칙:
- **원아**: 기본 배경 (회색 계열 테두리)
- **출석**: 초록 배경
- **투약 > 0건**: 주황 배경, 건수 표시
- **투약 0건**: 회색 배경, 0 표시

**클릭 동작**: 반 카드 클릭 → [화면 4] 오늘의 현황 모달 열림 (해당 반 데이터 기준)

---

**④ 오늘의 할 일 카드 (1fr 너비)**

기관관리자 할 일 목록 (6개):

| 항목 | 건수 | 클릭 이동 |
|------|------|-----------|
| 미작성 알림장 | 28건 | 알림장 페이지 |
| 가입 승인 대기 | 3건 | 가입 승인 관리 페이지 |
| 오늘 일정 | 3건 | 일정 관리 (준비 중 토스트) |
| 답변 대기 상담 | 2건 | 상담 관리 (준비 중 토스트) |
| 투약의뢰서 확인 | 4건 | 투약의뢰서 페이지 |
| 미발송 초대장 | 2건 | 초대 관리 페이지 |

- 각 항목: 색상 닷 + 항목명 + 건수 뱃지
- 전체 항목 클릭 가능 (cursor: pointer), 해당 페이지로 이동
- "준비 중인 기능입니다." 토스트: 일정 관리, 상담 관리

---

**⑤ 알림장 작성률 카드 (1fr 너비)**

- 상단: 완료 건수 / 미작성 건수 / 완료율 (%) — 3개 수치 박스
- 하단: 도넛 차트 (완료=파란색, 미작성=연한 빨간색), 범례 포함
- 반 필터 없음 → 항상 전체 반 합산 기준
- 수치는 CHILDREN_DATA 실시간 집계

---

**⑥ 최근 활동 카드**

- 제목: "최근 활동 전 교직원"
- 최근 4건만 미리보기 표시
- 각 항목: 아바타(이름 첫 글자) + 활동 내용 텍스트 + 경과 시간
- 항목 클릭 → 해당 콘텐츠 페이지 이동
  - 일정 관리 / 상담 관리: "준비 중인 기능입니다." 토스트
- 우측 상단 "더보기" 클릭 → [화면 5] 최근 활동 전체 모달 열림
- 대상: 전 교직원 (원장 포함), 반 구분 없음

---

**⑦ 공지사항 카드**

- 제목: "공지사항 전체 공지"
- 전체 공지 5건 리스트 표시
- 각 항목: 대상 태그 (전체/학부모/교사) + 제목 + 날짜
- 항목 클릭 → 공지사항 페이지 이동
- 우측 상단 "작성하기" 클릭 → 공지사항 페이지 이동
- 반 구분 없이 전체 공지 노출

태그 색상:
- 전체: 파란 배경
- 학부모: 초록 배경
- 교사: 주황 배경

## 화면 2: 대시보드 — 교사관리자 뷰

> 교사관리자 또는 통합관리자가 교사 컨텍스트로 접속한 경우.
> 담당 반 기준 원아 출석 현황 중심으로 표시.

---

**① 페이지 헤더**

- 좌측: "안녕하세요, [이름]님", 부제 "담당 반 오늘 현황을 확인하세요."
- 우측: 오늘 날짜 칩

**② 반 선택 필터바**

- **담당 반 1개**: 필터바 숨김, 해당 반 자동 고정
- **담당 반 2개 이상**: 셀렉트박스 표시
  - 담당 반: 정상 선택 가능
  - 비담당 반: 회색 처리 + 선택 불가 ("비담당" 표시)
  - 반 변경 시 오늘의 현황 카드 및 히어로 카드 즉시 업데이트

**③ 담당 반 히어로 카드 (교사 전용)**

- 담하 반 정보를 어두운 그라데이션 배경 카드로 강조 표시
- 좌측: 반 아이콘 + 반 이름(연령) + 날짜/기관명
- 우측: 전체 원아 수 / 오늘 출석 / 알림장 완료 / 미작성 4가지 수치

**④ KPI 카드 (4개 — 교사 전용 지표)**

| 순서 | 항목 | 수치 예시 | 보조 텍스트 |
|------|------|-----------|------------|
| 1 | 오늘 알림장 | 14/18 | 4건 미작성 |
| 2 | 출석률 | 89% | 결석 2명 |
| 3 | 미답변 상담 | 2 | 오늘 신규 1건 |
| 4 | 투약의뢰서 | 3 | 오늘 투약 예정 |

**⑤ 오늘의 현황 카드 (메인 카드)**

- 카드 제목: "오늘의 현황", 우측 액션 "전체 보기" 클릭 → [화면 3] 모달 열림

상단 출석 요약 (4개 박스):
| 구분 | 색상 |
|------|------|
| 출석 | 초록 |
| 결석 | 빨간 |
| 조퇴 | 주황 |
| 외출 | 파란 |

하단 원아 미리보기 테이블 (상위 5명):
- 컬럼: 원아명 / 출석 상태 / 알림장 / 투약
- 알림장: "✓ 완료" (초록) / "✗ 미작성" (빨간)
- 투약: "💊 N회" (주황) / "—"
- 원아 5명 초과 시 "+N명 더 보기 →" 클릭 가능 → [화면 3] 모달 열림

**⑥ 오늘의 할 일 카드 (교사 전용)**

| 항목 | 건수 | 클릭 이동 |
|------|------|-----------|
| 미작성 알림장 | 4건 | 알림장 페이지 |
| 투약의뢰서 확인 | 3건 | 투약의뢰서 페이지 |
| 미답변 상담 | 2건 | 상담 관리 (준비 중 토스트) |
| 오늘 일정 | 2건 | 일정 관리 (준비 중 토스트) |
| 학부모 초대 대기 | 1건 | 초대 관리 페이지 |

**⑦ 알림장 작성률 카드**

- 기관관리자 뷰와 동일 구성
- 수치: 현재 선택된 담당 반 기준 집계 (반 변경 시 즉시 업데이트)

**⑧ 최근 활동 카드 / ⑨ 공지사항 카드**

- 기관관리자 뷰와 동일 (반 구분 없이 전 교직원 활동, 전체 공지)

## 화면 3: 오늘의 현황 모달 — 교사 뷰 전체 보기

> 교사의 "전체 보기" 클릭 또는 "+N명 더 보기" 클릭 시 열림.
> 현재 선택된 담당 반 원아 전체 표시.

---

**① 모달 헤더**

- 제목: "오늘의 현황", 부제: "[반명] · 오늘" (예: "햇님반 · 오늘")
- 우측 ✕ 버튼으로 닫기 / 배경 클릭 닫기 / ESC 키 닫기

**② 요약 통계 (6개 박스 1행)**

| 항목 | 설명 |
|------|------|
| 출석 | 출석 인원 수 |
| 결석 | 결석 인원 수 |
| 조퇴 | 조퇴 인원 수 |
| 외출 | 외출 인원 수 |
| 투약 예정 | 당일 투약의뢰 있는 원아 수 |
| 알림장 미작성 | 알림장 미작성 원아 수 |

**③ 필터 탭 (4개)**

| 탭 | 표시 대상 |
|----|-----------|
| 전체 | 모든 원아 |
| 알림장 미작성 | `noticeWritten = false` |
| 💊 투약 있음 | `medicine ≠ null` |
| 결석·조퇴·외출 | `attend ≠ '출석'` |

- 탭 선택 즉시 테이블 필터링
- 한 번에 하나의 탭만 활성화

**④ 원아 테이블**

- 컬럼: # (순번) / 원아명 / 출석 상태 / 알림장 / 투약의뢰
- 모달 내 세로 스크롤 (최대 높이 88vh 고정)
- 테이블 헤더 고정 (sticky)

출석 상태 뱃지:

| 상태 | 색상 |
|------|------|
| 출석 | 초록 |
| 결석 | 빨간 |
| 조퇴 | 주황 |
| 외출 | 파란 |

셀 클릭 동작:

| 셀 | 클릭 결과 |
|----|-----------|
| 알림장 (미작성) | 알림장 페이지로 이동 (해당 원아 작성 모드 pre-fill) |
| 알림장 (완료) | 알림장 페이지로 이동 (해당 원아 조회 모드 pre-fill) |
| 투약 (있음) | 투약의뢰서 페이지로 이동 (해당 원아 pre-fill) |
| 출석 상태 | 토스트 메시지 표시 (전용 출석 관리 페이지 미구현) |

> ⚠️ pre-fill 연동: sessionStorage에 `noticePrefill` / `medicinePrefill` 저장 후 이동. 수신 페이지에서의 자동 처리는 미구현 (추후 연동 필요).

**⑤ 하단 힌트 텍스트**

"항목을 클릭하면 해당 원아의 기능으로 바로 이동합니다"

## 화면 4: 오늘의 현황 모달 — 반 카드 클릭 (기관관리자 뷰)

> 기관관리자 뷰에서 반별 현황 카드의 특정 반 클릭 시 열림.
> [화면 3]과 동일한 모달 구조. 클릭한 반 원아만 표시.

---

- 모달 부제: "[반명] · 오늘" (예: "새싹반 · 오늘")
- 표시 데이터: 해당 반 CHILDREN_DATA 기준
- 나머지 기능 [화면 3]과 동일

---

## 화면 5: 최근 활동 전체 모달

> 최근 활동 카드의 "더보기" 클릭 시 열림.
> 최근 1개월치 전 교직원 활동 이력.

---

**① 모달 헤더**

- 제목: "최근 활동 내역", 부제: "최근 1개월 · 전 교직원"
- ✕ 버튼 / 배경 클릭으로 닫기

**② 활동 목록 (날짜별 그룹)**

- 날짜 레이블로 구분 (예: "📅 오늘", "📅 3월 30일")
- 각 항목: 아바타(이름 첫 글자, 역할별 색상) + 활동 내용 + 역할 + 시각 + ↗ 이동 아이콘
- 항목 클릭 → 해당 콘텐츠 페이지 이동
  - 일정 관리 / 상담 관리: "준비 중인 기능입니다." 토스트
- 모달 크기: 가로 680px 고정, 세로 최대 640px (내부 스크롤)

목업 활동 데이터 기간: 3월 3일 ~ 3월 31일 (1개월)

활동 대상 교직원:

| 이름 | 역할 |
|------|------|
| 이하늘 | 교사 |
| 박민지 | 교사 |
| 최서연 | 교사 |
| 강지수 | 교사 |
| 김보라 | 교사 |
| 정다온 | 교사 |
| 김원장 | 원장 |
